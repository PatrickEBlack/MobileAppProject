How the App works:

This app works by prompting the user with a selection of all pokemon in the pokemon index. Found at "https://pokeapi.co/api/v2/pokemon?limit=12&offset=0". This link will provide the data for the first 12 pokemon in the index. It will then return the name of the pokemon which will be integrated into our buttons on the main page. Once a selection is made, the user will be brought to an external website which contains pictures and data on the respective pokemon. You are also able to navigate between 4 pages.
